# C200 矛盾解决报告 — Round 2 Verifier 反馈回复

## 矛盾 A: SP3000 饱和度悖论 ✅ 已解决

### 问题

- 像素侠实测: SP3000 ΔS = -0.160（比哈苏 X5 **低**）
- 社区共识: SP3000 = "色彩浓郁"
- R2 合并报告据此将 global_sat 从 1.06 下调至 0.95-1.02

### 独立验证结果

| 来源 | 描述 | 方向 |
|------|------|------|
| Zhihu 扫描仪对比文章 | "没有选用富士SP3000是因为，富士魔术手向来以胶片色出名——**色彩浓郁**" | SP3000 高饱和 |
| Zhihu 一次性相机回答 | "用富士SP3000扫出来**更黄了**"（色彩更浓） | SP3000 高饱和 |
| 豆瓣 Efinite Uxi 评测 | "用富士SP3000扫描出来的颜色会**浓郁一点**" | SP3000 高饱和 |
| smzdm 胶卷选择攻略 | SP3000 扫 Kodak ColorPlus "偏暖/浓郁"；哈苏扫同卷"偏青/颜色比较飘" | SP3000 高饱和 |
| smzdm 胶片扫描讨论 | "富士味太重...好多都是SP3000,这种扫描仪比较坑" | SP3000 风格化强 |
| **像素侠数据** | SP3000 vs **卖家校色后**的哈苏 X5 → ΔS = -0.160 | ❌ 参考系被污染 |

### 根因分析

像素侠分析的 9 组双扫对比来自新浪微博用户"摄影师故意城"（2019年2月3日发布）。其标题明确标注为"**哈苏X5（右，卖家校色）**"。

**卖家对哈苏扫描进行了手动色彩校正**——这在胶片冲扫行业是标准做法（"校色"）。这意味着哈苏参考样本的饱和度被人为提升到了卖家的审美偏好水平。SP3000 的"低饱和"是相对于一个被人为提高的基准而言的，不是绝对值。

这与社区共识（SP3000 = 浓郁色彩）一致：
- 不校色的 SP3000 直出 → 色彩较浓
- 手工校色的哈苏 X5 → 可以调至比 SP3000 更浓的饱和度

### 修正后的结论

**SP3000 扫描的 C200 呈现中等偏高的饱和度**，与社区共识一致。哈苏参考样本因卖家校色而被"过度饱和化"，不能作为 ground truth。

### 参数修正

| 参数 | R2 错误值 | R2.5 修正值 | 依据 |
|------|----------|------------|------|
| `global_saturation` | 0.95-1.02 | **1.06-1.10** | 社区多源共识（SP3000浓郁）+ Fuji X Weekly Color+2 映射 |

## 矛盾 B: 暖调数据 vs 冷调叙事 ✅ 已解决

### 问题的根源

这是一个**参照系混淆**问题。"暖/冷"是相对概念，三个独立参照系在 R2 中未被区分：

| 参照系 | C200 位置 | 证据 |
|--------|----------|------|
| **绝对中性**（R=G=B） | **微暖** — R/G=1.350, B/G=0.853 | 像素侠 7 张实拍像素测量 |
| **vs Kodak 家族** | **偏冷** — Gold 200 WB=+400K, C200 WB≈+200K | 社区共识（知乎/搜狐/豆瓣等多源一致） |
| **vs Fuji Superia 家族** | **中偏冷** — Superia 400 用 +R WB 暖化, C200 用 0R | Fuji X Weekly 配方系统对比 |

### 多源验证

**绝对暖的证据（像素侠）**:
- C200 实拍 7 张: R/G 均值 1.350, B/G 均值 0.853
- 中位色相 16°-39°（偏暖/橙）
- 这是 C-41 负片的橙色色罩 + 日光色温共同作用的结果

**相对冷的证据（社区共识）**:
- 知乎/Flickr/豆瓣/搜狐等多源一致描述"富士偏冷、柯达偏暖"
- Gold 200 = "阳光卷"（暖金）、C200 = "偏冷些"
- Fuji X Weekly: C200 配方 WB 0R/-3B → 在 Superia 家族中是**最接近中性**的配方
- 蔚然 D4 报告: C200 肤色"自然不推橙" → 对比 Gold 200 的暖金肤色

**SP3000 的角色**:
- SP3000 扫 Kodak → "更黄"（暖化）
- SP3000 扫 Fuji → 保持相对自然/微冷
- 社区"偏冷"认知部分来自 SP3000 对 Kodak 卷的额外暖化操作

### 统一叙事

> **C200 是一种温和暖调的胶卷，其温暖是克制的。在绝对意义上它偏暖（所有日光型 C-41 负片都因橙色色罩而偏暖），但在消费级彩色负片的光谱中，它处于冷端——比 Kodak Gold 200 的侵略性金琥珀暖调显著更冷，比 Fuji Superia 400/800 的刻意暖化更克制。C200 的"冷"不是物理学上的低色温，而是一种拒绝过度暖化的设计克制——自然肤色、自然蓝天、不推橙、不加蓝深度。这种克制在中国社区中被感知和描述为"偏冷"。**

### 修正后的设计哲学

```
C200 设计哲学：
  1. Gentle warmth, cool restraint — 暖而不烈，克制而非寒冷
  2. 自然肤色 — 不推橙，不猪肝红，保持皮肤本真色调
  3. 干净蓝天 — 不增强蓝色深度（CCB=Off），靠克制暖化来让蓝自然呈现
  4. 暗部暖红 — 阴天/暗部呈现温暖红紫倾向，增添故事感和怀旧氛围
  5. 柔和影调 — 中低对比度，柔和的光影过渡，保留高光细节
  6. 富士绿 — 偏黄绿、中高饱和、微亮的植物渲染，鲜明但不刺眼
```

### 参数修正

| 参数 | R2 值 | R2.5 修正值 | 依据 |
|------|-------|------------|------|
| `white_balance_shift_k` | +200K | **+200K**（维持） | 程远 -3B 映射 + 像素侠 R/G=1.350 确认微暖 |
| `shadow_tint` | [1.02, 0.99, 0.96] | **维持** | 程远引擎测试确认暗部红紫偏移再现 |
| `highlight_tint` | [0.99, 1.00, 1.02] | **[0.99, 1.00, 1.01]** | 微调 — C200 的"冷感知"不来自高光蓝 boost，而来自 warm suppression |
| `design_philosophy` | "cool signature" | **"gentle warmth, cool restraint"** | 统一暖/冷叙事 |

### 为什么保留 +200K 而不是负值

- 程远验证：Fuji WB -3B = 减蓝/加黄 = 暖化 → 对应引擎 +200K
- 像素侠验证：7 张实拍 R/G=1.350（明显暖）→ 如果是冷调，R/G 应 < 1.0
- C200 的"冷调感知"来自：CCB=Off（蓝色自然）、低 orange_push（4°）、中性肤色
- 如果错误地设置引擎 WB 为负值（冷化），会与实拍色彩数据冲突

## 综合修正后的 C200 参数 v3

```python
PRESET = {
    'name': 'c200',
    'title': 'Fujifilm Fujicolor C200 (Japan)',

    'tone': {
        'black_lift':                0.0015,
        'shadow_toe_pivot':          0.10,
        'shadow_toe_power':          0.88,         # 柔和暗部
        'contrast':                  0.96,         # 低对比度柔和影调（Fuji H+0.5/S-0.5 等效）
        'per_channel_contrast':      [1.00, 1.02, 1.00],  # G 通道微增（Fuji绿基础）
        'highlight_shoulder_start':  0.70,
        'highlight_shoulder_power':  1.20,
    },

    'color': {
        # 暗部暖红 — C200 的"故事感"签名
        'shadow_tint':             [1.02, 0.99, 0.96],     # 暗部 R↑/B↓ → 暖红偏移
        # 高光微蓝克制 — "冷"感知来自 warm suppression，而非 blue boost
        'highlight_tint':          [0.99, 1.00, 1.01],

        # 中等偏高饱和度 — SP3000 扫描特征 + Fuji Color+2 等效
        'global_saturation':        1.08,

        # 适度高光去饱和
        'highlight_desat_start':   0.65,
        'highlight_desat_max':     0.10,

        # 自然肤色 — 不推橙、不柔化
        'skin_hue_min':            12.0,
        'skin_hue_max':            35.0,
        'skin_sat_adjust':         1.00,

        # 克制的色相偏移 — C200 不风格化
        'teal_push':               2.0,
        'orange_push':             4.0,                    # 低于 Gold 200 的 10°

        # 自然蓝 — CCB=Off 等效
        'blue_saturation_boost':   1.02,
        'blue_luminance_shift':   -0.02,
        'blue_hue_shift':          0.0,                    # 不偏移（区别于 Leica +3° / Gold -5°）

        # 微暖白平衡 — 绝对暖调 + 相对冷调的统一
        'white_balance_shift_k': 200.0,
    },
}
```
