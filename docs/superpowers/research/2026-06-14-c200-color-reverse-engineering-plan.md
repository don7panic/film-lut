# 富士 C200（日产老版）色彩逆向工程 — 研究计划

**日期:** 2026-06-14
**方法:** deep-investigate（预计 10 轮探索）
**目标:** 老版日产 Fujifilm Fujicolor C200（日本制造、富士自有乳剂、已停产）

---

## 1. 核心问题

如何逆向工程**老版日产富士 C200** 胶片的色彩特征，将其参数化映射到 film LUT engine 的预设空间中，实现「复古纪实胶片感 — 光影柔和却充满故事感，每一帧都像电影画面」？

## 2. 目标画像（社区共识 + 初步搜索汇总）

| 维度 | 描述 | 证据强度 |
|------|------|---------|
| **整体色调** | 偏冷（相对于柯达偏暖），富士系典型冷调 | 中 — 多源社区共识 |
| **暗部色彩** | "富士胶卷特有的暗部绿色" — 暗部有绿色偏移 | 中 — 搜狐/社区描述 |
| **绿色渲染** | "富士绿" — 饱和度较高，略有塑料感 | 中 — 搜狐胶卷攻略 |
| **肤色表现** | 能"生动再现亚洲人的肤色"，阴天比柯达稍好 | 低 — 营销文案 + 社区 |
| **阴天表现** | "发红发紫" — 阴天暗部偏红紫 | 中 — 社区共识 |
| **饱和度** | 色彩稍重，介于 Velvia（浓艳）和 Provia（清淡）之间 | 低 — 爱问知识人 |
| **对比度/影调** | "光影柔和却充满故事感" — 柔和但非平淡 | 低 — 用户主观描述 |
| **适用场景** | 晴天和雨天表现好，阴天表现一般 | 中 — 社区共识 |

## 5. 限制声明

1. **C200 技术数据表可能不可获取** — 老版日产 C200 已停产，Fujifilm 官网可能已移除数据
2. **无 ColorChecker + C200 实测数据** — 网上无公开的色度学标定研究
3. **样张分析受扫描仪/冲扫店影响极大** — SP3000 vs 哈苏 vs 诺日士扫描结果差异巨大
4. **Fuji X Weekly 配方是 Superia 系列，非 C200** — 参照物本身不同，需推断映射
5. **Analogica Lab LUT 是商业产品** — 可作为对照但不可作为 ground truth
