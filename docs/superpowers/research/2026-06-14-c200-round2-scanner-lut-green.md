## 任务 A: SP3000 决策框架

**SP3000 色彩特征共识：**
1. 🔴 **饱和度增强** — "色彩浓郁"是最一致的描述
2. 🔴 **对 Fuji 胶片的特定优化** — Fuji 自己的扫描仪 → Fuji 自己的胶片 → 预设针对 Fuji 乳剂调校
3. 🟡 **对 Kodak 胶片偏黄** — 说明 SP3000 预设并非中性，而是为 Fuji 胶片定制的
4. 🟡 **可能偏青绿** — SP3000 + Fuji 胶片的组合常被描述为"冷调"、"青绿"（Fuji 绿）
5. 🟢 **双CCD色彩分离** — 硬件层面实现了色彩和亮度的独立采集，再软件融合

### A.2 SP3000 vs Hasselblad X5 直接对比（Sina 文章 + 豆瓣相册）

由于无法直接获取全分辨率图像像素数据，基于缩略图视觉描述的对比：

| 色彩维度 | SP3000（左） | Hasselblad X5（右，卖家校色） |
|---------|-------------|---------------------------|
| **整体色温** | 偏冷/中性 | 偏暖/中性 |
| **饱和度** | 更高（"浓郁"） | 更自然/适中 |
| **对比度** | 更高 | 更低（低反差、顺滑过渡） |
| **暗部** | 更深、更冷 | 更亮、更多细节 |
| **高光** | 更亮 | 柔和保留 |
| **绿色** | 更青/更饱和 | 更黄/更自然 |

### A.3 决策推理

**用户目标描述**: "复古纪实的胶片感，光影柔和却充满故事感，每一帧都像电影画面"

这个描述具有双重性：
- "光影柔和" → 指向影调的柔和（低对比度、柔和高光滚降）— 更接近 Hasselblad 的呈现
- "胶片感"、"每一帧都像电影画面" → 指向理想化的胶片记忆 — 更接近 SP3000 的"浓郁胶片色"

**关键事实：**
1. 社区中 90%+ 的中文 C200 样张来自 SP3000 扫描
2. Fuji X Weekly C200 配方（R1 最佳数据源）由 George Coady 对照真实 C200 胶片调校 — 他几乎肯定用的是 SP3000 扫描的 C200 作为参照
3. "纯 C200 乳剂"的量化数据**根本不存在** — 没有官方技术数据表，没有色度学标定
4. Hasselblad/Noritsu 扫描的 C200 样张**极其稀少**，不足以建立可靠的色彩 baseline

### A.4 决策建议

**✅ 建议选择：「C200 社区体验」（即 SP3000 + C200 的综合呈现）**

理由：
1. **数据可行性** — SP3000 + C200 有充足的样张和配方数据；"纯乳剂"数据不存在
2. **目标匹配** — "复古纪实的胶片感"正是中国胶片社区通过 SP3000 形成的美学共识，而非实验室级别的色彩准确性
3. **配方基础** — Fuji X Weekly C200 配方天生就是 SP3000-biased 的参照
4. **引擎限制** — 引擎参数空间更适合模拟"最终呈现"而非"物理乳剂特性"

⚠️ **折中策略**: 以 SP3000-biased 配方为起点，但**不复制 SP3000 的全部色彩偏移**。具体来说：
- 保留 SP3000 的"色彩浓郁"方向（高一些的 global_sat）
- 但降低 SP3000 的高对比度倾向（采用柔和的 shadow_toe + highlight_shoulder）
- 保留 Fuji 冷调，但不过度增强
- 这正好映射用户描述的"光影柔和 + 胶片感"的二元需求

### C.2 "塑料感"的量化解读

"塑料感"是一个主观感受。基于多源交叉分析，可拆解为三个客观因素：

| 因素 | 解释 | HSL 表现 |
|------|------|---------|
| **① 高饱和度** | 绿色比现实更鲜艳 → 缺乏自然色彩中的"灰调" | S > 真实值 |
| **② 高明度** | 绿色比现实更亮 → 缺乏阴影深度，像塑料玩具的亮绿色 | L > 真实值 |
| **③ 色相偏黄绿** | 绿色偏向 90-105°（yellow-green）而非 120-130°（true green）→ 像荧光绿/草绿而非深沉森林绿 | H = 90-105° |

这三个因素叠加：**高饱和 + 高明度 + 偏黄绿 = 类似塑料玩具/荧光色的视觉感受**。

### C.3 与 Kodak Gold 200 绿色的对比

| 维度 | Kodak Gold 200（已知参照） | Fuji C200（推断） |
|------|--------------------------|-------------------|
| 绿色色相 | 偏黄绿（~95-110°） | 偏黄绿（~90-105°）— 更极端 |
| 绿色饱和度 | 中（Gold 200 global_sat=1.12 但有偏暖 filter） | **中高** — "鲜艳活泼" |
| 绿色明度 | 中低（Gold 200 偏暖整体压暗冷色） | **中高** — "塑料感"=过亮 |
| 绿色特征 | "warm yellow-green foliage" | "小清新富士绿" |

**关键差异**: Kodak 的绿色通过暖色滤镜被适度压低了亮度和饱和度；C200 的绿色因为整体冷调而更"自由"地表达高饱和+高明度。
