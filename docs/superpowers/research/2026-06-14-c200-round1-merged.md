## 🔥 关键发现（按重要性排序）

### 1. 直接对标 C200 的公开配方（雅婷）
- **来源**: Fuji X Weekly, George Coady 对照真实 C200 胶片调校
- **Film Simulation**: Classic Negative
- **核心参数**: Highlight +0.5 / Shadow -0.5 / Color +2 / WB Daylight 0R,-3B
- **CCB = Off**: C200 是 Superia 家族中唯一不增强蓝色深度的配方
- **含义**: C200 的冷调来自 WB 偏移（-3B），而非蓝色通道增强

### 2. 暗部色彩是红紫色，非绿色（思源 + 志远）
- **矛盾来源**: 搜狐创意写真文章称"富士胶卷特有的暗部绿色"→ 可能是 Superia/Reala 系列特征，不适用于 C200
- **C200 实际暗部**: 多源独立确认阴天暗部"发红发紫"
- **可能原因**: C-41 橙色色罩去色罩不完美 + SP3000 富士通道预设
- **影响**: `shadow_tint` 应从"G boost"改为"R slight boost + B slight cut"

### 3. SP3000 扫描仪预设可能定义了社区认知的 C200（志远）
- SP3000 → 浓郁、偏冷/青绿、高对比 → 这定义了社区中的"C200 look"
- 诺日士 → 中性、真实还原
- 尼康 9000ED → 最接近乳剂本真
- **关键决策点**: 复刻目标是"纯 C200 乳剂"还是"C200 + SP3000 综合体验"？

### 4. C200 的 Fuji 绿（思源 + 志远）
- 方向: 偏黄绿（90-110°），非深绿（120-130°）
- 饱和度: 较高（"富士绿有点明显"）
- 质感: "偏塑料感"（可能 = 高 saturation + 高 luminance 导致缺乏深度）
- 引擎缺口: 需要新增 `green_saturation_boost`、`green_hue_shift`、`green_luminance_shift`

### 5. C200 族谱确认（明伟）
- 3 层乳剂结构（非 Superia 的 4 层）→ 绿/青再现能力弱于 Superia
- 老版日产已停产，新版美产曲线 ≈ Kodak Gold 200（非本次目标）
- 官方技术数据表不存在于公共领域
