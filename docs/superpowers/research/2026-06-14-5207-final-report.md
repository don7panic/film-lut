## 1. 5207 色彩 DNA

### 设计哲学

*5207 是一种中性偏微冷的日光电影负片。它的「中性」不是平淡——而是高色彩分离度下的自然呈现：皮肤不推橙、蓝天有力度、阴影干净中性。它的极端宽容度（-5.4 到 +7 档）意味着在任何光线下都能捕捉最动人的瞬间。*

## 4. Standard vs V-Log 决策

| 路径 | 适用场景 | 阴影质量 | 推荐度 |
|------|---------|---------|--------|
| **Standard LUT** | S9 Standard (Rec.709) 拍摄 | ✅ 完整分辨率 | 🔴 **默认推荐** |
| V-Log LUT | S9 V-Log 拍摄 | ❌ 12.5% 阴影丢失 | 🟡 高级用户可选 |

**根因：** V-Log 编码中 V < 0.125 的码值全部映射到 scene-linear 0，33³ 均匀采样导致前 4 个采样点无阴影信息。这是 V-Log 编码的固有限制，非引擎 bug。

**预设设置：** `skip_vlog = True`（与 5219 一致）

## 6. 使用方式

```bash
cd /Users/oolong/workspace/film

# 生成 Standard LUT（唯一输出，skip_vlog=True）
uv run python lutex.py --preset 5207

# 生成 LUT + 测试预览图
uv run python lutex.py --preset 5207 --preview

# 应用到 Standard 照片
uv run python lutex.py --preset 5207 --apply your_photo.jpg

# 应用到 V-Log 照片（需先临时修改预设 skip_vlog=False）
uv run python lutex.py --preset 5207 --apply your_vlog_photo.jpg --apply-type vlog
```

将 `luts/S9_Standard_to_5207.cube` 复制到 S9 SD 卡的 LUT 目录即可在机内加载。
