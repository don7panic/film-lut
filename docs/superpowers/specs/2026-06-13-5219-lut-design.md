# Kodak 5219 Film Emulation LUT for Panasonic S9

**Date:** 2026-06-13
**Status:** Design

## Overview

Create `.cube` LUT files for Panasonic S9 camera that emulate the look of Kodak Vision3 5219 (500T) color negative film. The LUTs are designed for in-camera loading, daylight-balanced (5600K reference), with moderate intensity that captures the signature 5219 aesthetic while remaining versatile and natural.

## Outputs

| File | Input | Description |
|------|-------|-------------|
| `S9_VLog_to_5219.cube` | V-Log / V-Gamut | Full pipeline: gamut mapping → tone curve → 5219 grade |
| `S9_Standard_to_5219.cube` | Standard (Rec.709) | Creative grade only (Rec.709 → 5219 look) |
| `generate_lut.py` | — | Python script that generates both LUTs with tunable parameters |
| `test_reference.png` | — | Reference image with before/after comparison |

## Color Pipeline

### For V-Log input:
```
V-Log / V-Gamut
  → Gamut: V-Gamut → Rec.709 (matrix transform)
  → Gamma: V-Log → Scene-linear → S-curve (1D + 3D)
  → Creative: 5219 tonal/color adjustments (3D LUT)
  → Rec.709 display
```

### For Standard input:
```
Rec.709
  → Creative: 5219 tonal/color adjustments (3D LUT)
  → Rec.709 display
```

## Design Parameters

### Tone Curve (S-Curve)
- **Black lift:** 0.02 (Ire) — prevents crushed blacks, mimics film base density
- **White roll-off:** soft knee starting at 0.85, reaching 1.0 at input 1.0
- **Mid-gray mapping:** 0.45 → 0.42 (slight brightening, lower contrast in midtones)
- **Overall contrast boost:** ~15-20% S-curve amplitude relative to linear
- **Toe:** gentle compression below 0.10 (shadow detail preservation)
- **Shoulder:** gradual compression above 0.80 (highlight protection)

### Color Matrix
- **Skin tone protection** — hue range 15°–30° preserved, saturation reduced 5-10% for smoothness
- **Shadow color bias** — cyan/blue shift equivalent to -500K in shadows, +300K in highlights
- **Teal-orange palette** — subtle push of blues toward teal (hue ~190° → ~200°), warm tones toward amber (~30° → ~35°)
- **Global saturation** — reduced 5-8% overall
- **Highlight desaturation** — progressive desaturation above 0.70 luminance

### 5219 Signature Characteristics Modeled
1. Warm golden skin tones with smooth highlight roll-off
2. Cyan-biased shadows (the "film shadow" look)
3. Moderate contrast with soft highlight compression
4. Slightly desaturated overall with selective color emphasis
5. Warm/cool split-tone (teal-orange complementary)

## LUT Format

- **Format:** `.cube` (Iridas/Resolve compatible)
- **Size:** 33³ (35,937 entries) — balance of accuracy and file size for in-camera use
- **Domain:** 0.0–1.0 (normalized RGB)
- **Input range:** full range (0–1)

## Phase A: Hand-Crafted Generation

Python script (`generate_lut.py`) that:

1. Defines tunable parameters as constants at the top of the file
2. Generates a 3D LUT grid (33×33×33)
3. Applies transforms in order: gamut conversion (V-Log only) → Log-to-Linear gamma → S-curve → creative color → Linear-to-sRGB gamma
4. Writes valid `.cube` files with proper headers
5. Includes a `--preview` mode to generate before/after test images

### Dependencies
- Python 3.9+
- NumPy (meshgrid, linear algebra for gamut transforms)
- Pillow (optional, for preview images)

## Phase B: Research & Calibration

After Phase A produces working LUTs, research publicly available 5219 color data:

1. Search for Kodak 5219 spectral sensitivity/dye density curves
2. Look for ACES IDT/ODT references for 5219
3. Find 5219 film scan samples for visual comparison
4. Compare Phase A LUT output against reference data
5. Adjust parameters based on measured deviations
6. Generate calibrated LUT versions: `S9_VLog_to_5219_v2.cube`, `S9_Standard_to_5219_v2.cube`

## Constraints

- Must be single-file LUT (no chain) for in-camera compatibility
- Must fit within S9 LUT memory limits (33³ is safe)
- Must not crush shadows or clip highlights excessively (shooting LUT — needs to work across varied scenes)
- Input/output in Rec.709 display space for proper monitoring

## Testing

- Generate synthetic color ramp/gradient test images
- Apply LUT and verify: no banding, no out-of-gamut clipping, smooth gradients
- Visual check: skin tones natural, shadows not crushed, highlights not blown
